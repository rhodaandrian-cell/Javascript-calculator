# JavaScript Calculator

This is a simple calculator built with JavaScript that can perform addition, subtraction, multiplication, division, square root and power function. 
It also keeps a history of all calculations.

## How to Run

1. Git clone the repository
2. Input  `code .` in your terminal to open VS Code editor
3. Open `calculator.js` in your code editor.
4. Run the file with Node.js:
5. Check the console for calculation results and history.
6. History shows the last calculation provided, to avoid too much number statements after a long period of time usage.

## Features

- Addition, Subtraction, Multiplication, Division, Square root, Power Function
- Calculation history tracking
